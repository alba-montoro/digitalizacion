# 🏢 Proyecto Odoo + Facturación + Sitio web 

He creado una compañía ficticia de jardinería llamada "Mycompany". En ella he creado dos clientes (una empresa ficticia y una persona ficticia) y tres productos (dos bienes y un sevicio). Para ello he utilizado los módulos "Facturación" y "Sitio web" disponibles en Odoo.

## 🛠️ Tecnologías utilizadas

* **Odoo**
* **Base de Datos:** PostgreSQL (por defecto)
* **Contenedores:** Docker & Docker Compose
